Этапы создания сайта

Выбор макета

Первый этап разработка сайта это выбор шаблона. Создание сайта с помощью макета это очень удобно, легко и главное качественно.

Написание кода

Второй этап создание сайта это написание кода. Используется языки как HTML, CSS и JS. В качестве допольнительного ресурса входить некоторые плагины.

Хостинг

Финалный этап создание сайта это хостинг. Если по другому сказать добавим сайт в интернет.

Шаблоны

Для создание сайта потребуется шаблоны. Бывает 2 тыпа шаблонов. На заказ и готовый. Готовые шаблоны это готовые бесплатные шаблоны которые можно найти в приложении Figma. А на заказ это те шаблоны которую нужно заказать на дизайнера и оплатить доподьнителную денег.

Написание кода

Второй этап создание сайта это написание кода. Этот этап является самой главной часть создание сайта потому что мы создадим сайт. Для написание кода нужно иметь работать с языками программирование как HTML, CSS и JS.

HTML один из самых популярных языков программирование. Этот язык программирование нужно чтобы добавить блоки на сайт например: параграфи(Р), заголовки(Н) и т.д.

CSS это язык стилей. Используя этот язык мы можем стилизировать определённый блок. Например:мы можем изменить фон или задать ширину и высоту блока, задать цвет к блоку и т.д.

JS это одиниз часто исползуемих языков программирование. С помощью этого языка мы можем создать гамбургер, слайдер, анимация при скролле, эффект при клике мыщью и многое чего.

Плагины

Для быстрого и эффективного создание сайта можно исползовать плагины как Bootstrap. Быстрий дизайн, адаптивний вёрстка, готовые коды и многое другое можно найти в bootstrap.

Необычний дизайн

Для более красивого страници мы исползуем белое и чёрное блоки. Ещё мы исползуем допольнителние шрифти и подходяший свет кнопок

Хостинг и домен

Посдедный этап создание сайта это хостинг и домен. Эсли по другому сказать добавим сайт в интернет. Домен это имя или ссылка сайта например: web-site.com Бывает 2 типа домена: бесплатные и платные. Бесплатные web-site.github.io а платные web-site.com, web-site.io и т.д
